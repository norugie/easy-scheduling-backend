# easy-scheduling-backend

Node, Express, TypeScript, Drizzle, and PostgreSQL boilerplate.

## Environment

- `PORT=3000`
- `DATABASE_URL=postgres://postgres:postgres@localhost:5432/easy_scheduling`
- `CORS_ORIGIN=*`
- `DB_POOL_MAX=10`
- `DB_IDLE_TIMEOUT_MS=30000`
- `DB_CONNECTION_TIMEOUT_MS=2000`

## Scripts

- `pnpm dev`
- `pnpm build`
- `pnpm start`
- `pnpm typecheck`
- `pnpm self-check`
- `pnpm db:generate`
- `pnpm db:migrate`
